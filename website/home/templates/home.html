Note Maker


This site provides an intuitive way to consolidate class notes in preparation for revision. The techniques that are implemented here work well for me, but are obviously unlikely to suit everyone. The site works by first allowing you to enter notes for different courses, modules, and headings. Each "point" can have "sub-points", allowing for nested concepts. When it comes to revising, you can choose to be presented with headings that you then have to fill out the relevant points under, or you can pre-define some "fill-the-gap" notes to complete. Each attempt is tracked, so you can view how many times you have reviewed each point, heading, and module.
